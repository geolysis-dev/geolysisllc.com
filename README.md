This repo contains the source code for [geolysisllc.com](https://geolysisllc.com).

## Local development

```shell
# Download the repo
git clone https://github.com/geolysis-dev/geolysisllc.com.git

# Enter the repo
cd geolysisllc.com

# Install the dependencies
npm install

# Start local development
npm run dev
```
